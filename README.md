#  File Organizer using Python (os module)

This project is a simple Python script that automatically organizes files in a directory based on their file types (extensions).  
It helps keep folders clean by sorting files into categorized subfolders such as Images, Documents, Videos, etc.

---

##  Features

-  Automatically organizes files by extension  
-  Creates folders if they don’t exist (optional) 
-  Cleans up messy directories  
-  Lightweight and fast  
-  Built using Python’s built-in os module  

---

##  Tech Stack

- Python
- os module
